# 朱砂操盘台 · 小红书发布助手（Chrome 插件）

用**你本机已登录的小红书**，在朱砂操盘台里：
1. **抓真实对标笔记**（流水线「找对标」）——用你自己的账号 + 住宅 IP 抓，**不限次、不弹验证码、不依赖服务器共享账号**
2. **一键存草稿**——把成稿存进小红书草稿箱，和你手动发一模一样，**不触发风控**

全程在你本机浏览器、用你真实 IP / 真实会话，数据只回传到 yonglin.chat 页面，不上传任何第三方。

## 安装（一次）
1. Chrome 打开 `chrome://extensions/`
2. 右上角打开 **「开发者模式」**
3. 点 **「加载已解压的扩展程序」**，选这个 `extension/` 文件夹
4. 安装后地址栏右侧会出现插件图标，点开应显示「✓ 插件已安装」

## 用法 A：抓真实对标（流水线「找对标」）
1. 在 Chrome 里**登录小红书**（打开 https://www.xiaohongshu.com 确认已登录）
2. 打开 **yonglin.chat → 流水线 → 第二步 对标拆解**
3. 点 **「📥 抓约20篇对标」** → 插件在后台用你的账号抓真实笔记，回填到页面（不限次）

## 用法 B：一键存草稿（一键发布）
1. 在 Chrome 里**登录小红书**（打开 https://creator.xiaohongshu.com 确认已登录，要发哪个号就登哪个）
2. 打开 **yonglin.chat → 一键发布**，选一篇成稿
3. 点 **「🧩 用插件发布到草稿箱」**
4. 插件自动打开小红书创作页、传图、填标题正文 → **你在该页确认「存草稿」**（终审/发布由你把关，合规）

## 多账号
浏览器里切换登录的小红书账号即可，插件跟当前登录账号走。

## 说明
- 小红书创作页改版时，自动填充的选择器可能要更新——反馈给开发即可。
- 图片：朱砂会把封面/长图笔记卡渲染成 PNG 一起带过来。
- 这是发布到**草稿箱**，不自动公开发布。
