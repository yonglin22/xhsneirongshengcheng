朱砂 · 小红书发布助手
本机浏览器发布，零风控
📱 检测到安卓/移动端
请在浏览器菜单里开启「桌面版网站 / Desktop site」，再打开小红书创作中心，插件才能正常抓取/发布（移动版页面结构不同）。
🆕 有新版可更新
下载后解压，到 chrome://extensions 点本插件的「重新加载」即可更新。
✓ 插件已安装
  1. 在 Chrome 里登录小红书（creator.xiaohongshu.com）
  2. 打开 yonglin.chat → 一键发布
  3. 选成稿 → 点「🧩 用插件发布」
  4. 会自动打开创作页填好内容 → 你确认「存草稿」
用你已登录的小红书操作，和手动发一样，不触发风控。
📡 本机作为执行设备
开启后，本机会自动领取你在「获客计划」里下发的任务并执行。
👀 前台观看模式
开启后养号标签前台打开，你能实时看着它在小红书里滑动/点赞；关则后台静默跑。
📋 获客计划（养号/截流）
直接执行你在 yonglin.chat 「获客计划」里配置好的计划，含好感率/关注/评论/截流引流/话术来源等完整配置。
💌 待发私信草稿（人工确认）
私信不自动发，AI 仅生成话术草稿；你确认要发再手动去对方主页发，规避高风险灰色操作。
