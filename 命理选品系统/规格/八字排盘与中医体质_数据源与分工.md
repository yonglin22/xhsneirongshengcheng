## 二、需要你提供（这几样我造不了权威版）

### ① 香承真实香方库（最关键）
每个在产香方给：**名称 · 君臣佐使配伍 · 功用 · 对应 SKU · 价格 · 现货/定制 · 周期**。
> 我现在写的"温阳养气 / 解郁凝神 / 滋阴安神 / 化湿醒神 / 养气温肺"是**占位**，要换成香承实际在产的。你有 PDF，整理成一张表给我即可。

### ② 中医专业审定的「体质 → 香方 / 石头」对应表
我做的"五行→体质→香方"是**工程桥接**；但"哪种体质配哪个香方、哪颗石头"应由**懂中医的人**（香承香师 / 你们合作的中医师）审定签字——避免乱配、避免夸大功效。这是可信度与合规的根。
> 交付物：一张 9 行的表（平和/气虚/阳虚/阴虚/痰湿/湿热/血瘀/气郁/特禀 → 推荐香方 + 主石 + 一句养生说明）。

### ③ 体质判定方式（产品决策，你拍板）
- **A. 纯八字五行桥接**（我已做）：零额外步骤、不打扰用户；但体质是"倾向"非"辨识"。
- **B. 九型体质问卷**：官方《中医体质分类与判定》（王琦九种体质，中华中医药学会 **ZYYXH/T157-2009**，标准 60 题）或自研简版（如 27 题快测）。**官方量表有标准/版权**，需你决定用官方版还是自研并做合规。给我题目 + 评分规则，我做问卷 UI + 自动判定 + 写进向量。
> 建议：**先 A 上线**（最快最省），**B 做二期精修**（问卷体验感 + 更准）。

### ④ 合规红线 / 受限词表（+ 法务过一遍）
功用文案统一走"养生调理 / 气味体验 / 传统香方灵感"，**不诊断、不治疗、不夸大、不承诺疗效**。你给合规底线（或让法务审），我把话术钉进产品与 SKU 文案。

### ⑤（可选·加分）真实体感反馈数据
体质—香方的用户体感反馈，用来校准映射权重。
