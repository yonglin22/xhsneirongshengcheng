## 1. 时间线记忆（最容易爆，直接对标已验证爆款）
- **选题角度**：把《世界现代设计史》按时间轴串成一张图，解决“记不住流派顺序”。
- **标题**：
  - `一张图记住《世界现代设计史》时间线，背了3年终于通了`
  - `世界现代设计史时间线丨从工艺美术运动到后现代，一图通关`
- **正文结构**：痛点(流派乱、年份混) → 一张时间轴图 → 3个记忆锚点(口诀) → 领取完整版。
- **配图**：长文笔记图，横向时间轴 + 每个节点色块。

## 2. 五大类型万能句（论述题提分）
- **选题角度**：论述题答不满？给 5 类“万能句式模板”。
- **标题**：
  - `设计史论述题5大万能句，直接套，从12分到20分`
  - `考前必背丨设计史论述“起承转合”模板，谁用谁高分`
- **正文结构**：论述失分点 → 5类句式(背景句/影响句/评价句/对比句/升华句)各1例 → 领全套。
- **配图**：白底干货卡，每类一句范例+填空位。

## 3. 10个主要国家设计特点（考前冲刺）
- **选题角度**：中/美/英/德/日…各国设计特点带背，覆盖高频论述。
- **标题**：
  - `设计史丨10个主要国家设计特点，一次背完（考前必看）`
  - `中美英德日…10国设计风格对照表，论述题直接抄`
- **正文结构**：为什么按国家背 → 表格式10国特点关键词 → 记忆口诀 → 领取。
- **配图**：对照表长图。

## 4. 手绘快题时间分配
- **选题角度**：3小时快题怎么分配不画崩。
- **标题**：
  - `设计手绘快题3小时时间表，照着来不会画不完`
  - `快题总画不完？这张时间分配表救了我`
- **正文结构**：画崩的真实原因(前松后紧) → 分钟级时间表 → 3个提速技巧 → 领模板。
- **配图**：时间条形图 + 快题排版示意。

## 5. 院校选择避坑
- **选题角度**：设计考研怎么选校，避开“大小年/压分”坑。
- **标题**：
  - `设计考研选校避坑丨这4类院校慎报，我踩过的坑别再踩`
  - `二本想上岸设计研，这样选校成功率翻倍`
- **正文结构**：选校3个致命误区 → 4类院校特征 → 自测清单 → 领院校数据表。
- **配图**：决策清单卡。

## 6. 背诵计划表（暑期/冲刺）
- **选题角度**：史论那么多，什么时候背、背几轮。
- **标题**：
  - `设计史论背诵计划表丨暑假到考前3轮怎么安排`
  - `史论背了就忘？按这张表背3轮，考场默出来`
- **正文结构**：遗忘曲线痛点 → 3轮时间表 → 每轮任务 → 领可打印版。
- **配图**：日历式计划长图。

## 7. 名词解释答题公式
- **选题角度**：名词解释怎么答满分不废话。
- **标题**：
  - `设计史名词解释“3句公式”，一个15分稳拿`
  - `名词解释别乱写！这个公式判卷老师最爱`
- **正文结构**：低分示范 vs 高分示范 → 3句公式(定义+背景+影响/评价) → 例子 → 领100词。
- **配图**：对比卡(错误❌/正确✅)。

## 8. 高频考点自测清单
- **选题角度**：给一张“会不会”自测表，制造互动。
- **标题**：
  - `设计考研史论高频考点自测，敢说全会的没几个`
  - `40个设计史高频考点，能默出30个稳上岸`
- **正文结构**：自测玩法 → 清单(打钩) → 结果解读 → 领完整清单+答案。
- **配图**：checklist 长图（互动性强、易收藏）。

## 9. 中外工艺美术史对照
- **选题角度**：中外工美史容易混，做对照。
- **标题**：
  - `中外工艺美术史对照表，再也不会张冠李戴`
  - `工美史丨中国VS外国，一张表理清时间线和代表作`
- **正文结构**：混淆痛点 → 左右对照表 → 3个记忆钩子 → 领取。
- **配图**：左右分栏对照长图。

## 10. “我怎么1天做出一周的笔记”（工具种草，转化最强）
- **选题角度**：直接讲你怎么用 AI 3分钟出一张考研干货图——这条是给“也想做号/也在考研想省时间”的人看的，转化工具最直接。
- **标题**：
  - `我用AI把设计考研资料“嚼碎”成笔记图，1天顶一周`
  - `考研人必看丨我怎么3分钟做一张能背的设计史干货图`
- **正文结构**：以前手动整理多痛苦 → 现在填个方向AI出图(录屏/截图) → 效果对比 → 想要同款工具+模板私我。
